# inventory-rules

Reglas duras del proyecto app-inventory.

- R1: UI responsiva 2D (ancho y alto), botones de accion siempre visibles.
- R2: permisos por constante, centralizados, no strings sueltos.
- R3: auditoria sensible detras de un PIN.
- R4: CHANGELOG obligatorio en cada release.
